LST Processing Tool

Summary

The LST Processing Tool allows to calculate Land Surface Temperatures from a synergetic approach, given overlapping MERIS/AATSR Level 1b product. This tool also provides a simulation mode which supports as input selected OLCI/SLSTR L1 simulation datasets. The processor was developed in the frame of the ESA Sen4LST project.

Algorithms

The underlying algorithms are presented in detail in the corresponding ATBDs [1], [2], and summarized in the Processor Technical Specification document [3].
